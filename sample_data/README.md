# Sample data

This folder is reserved for small, synthetic sample data.

Do not add real project data, private geodata, sensitive nature observations, municipal data, or large raster files.

Recommended future sample dataset:

```text
sample_data/
├── synthetic_classes.tif
├── synthetic_roi.gpkg
└── README.md
```

The sample should be simple enough to manually verify expected histogram results.

If no safe sample data exists yet, keep only this README.
